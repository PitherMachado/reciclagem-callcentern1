# Portal N1 — Treinamento Empréstimo Pessoal INSS

Projeto estático pronto para subir no GitHub.

## Estrutura

- `index.html`
- `style.css`
- `script.js`
- `assets/bg.png`

## Como usar

1. Envie todos os arquivos para um repositório no GitHub.
2. Abra o `index.html` localmente ou publique via GitHub Pages.
3. Para GitHub Pages:
   - Settings
   - Pages
   - Deploy from a branch
   - Selecione a branch principal e a pasta raiz

## Controles

- Botões na interface
- Teclas `←` e `→`
- Tecla `A` abre a agenda

## Objetivo

Treinamento premium N1 com foco em:

- domínio de produto
- validação operacional
- abordagem consultiva
- objeções
- conversão
